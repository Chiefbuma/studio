# **App Name**: Cake Paradise

## Core Features:

- Component Conversion: Convert existing JSX components to TSX to leverage TypeScript's type safety.
- Componentization: Break down the application into reusable, testable components following software design principles.
- MS SQL Integration: Implement connection with MS SQL to handle persistence.
- Express Backend: Create an Express backend with API endpoints to interact with the database.
- Unit Tests: Write unit tests for backend logic.
- AI Recommendation Tool: Provide cake recommendations powered by AI that takes into account user history

## Style Guidelines:

- Primary color: Deep amber (#C68324) to reflect warmth and appetite.
- Background color: Very light desaturated amber (#FAF5EB) as a soft, warm background.
- Accent color: Mustard yellow (#D4A743) for highlighting interactive elements.
- Headline font: 'Playfair' serif for headlines
- Body font: 'Inter' sans-serif for body
- Code font: 'Source Code Pro' monospace font for inline code snippets, not the app font. 
- Use high-resolution cake and sweets-related icons from FontAwesome.
